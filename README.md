# CPL Instructor App

This repository provides the baseline Azure infrastructure for the CPL Conversational AI deployment.

Stable infrastructure release:
v1.0-stable-infra

Student Teams must sync to this version before continuing development.

1- Go to your fork
2- Click “Sync fork”
3- Update branch
4- Confirm you are aligned with tag v1.0-stable-infra

Do not modify:
- startup.sh
- requirements.txt
- deployment workflow
- environment variable naming
